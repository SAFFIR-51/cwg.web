# CWG ReturnValue — 새 카피 드래프트 (광고주 대상 B2B 톤)

> **목적**: 앱에 들어갈 광고주를 모집하는 B2B 사이트.
> **톤 핵심**: "로또 번호 추천 회사" 가 아닌 **"로또를 콘텐츠로 만드는 회사"**.
> 특허/데이터는 자랑이 아니라 **광고주가 안심할 이유**로 풀어냄.
>
> 지원 로또 4개 = 한국 동행 6/45 · 일본 로토6 · 유럽 EuroMillions · Eurojackpot
